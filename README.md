# 🎬 Movie Review Sentiment Analysis Using NLP and Streamlit

## 📌 Project Overview
This project is a Movie Review Sentiment Analysis web application built using Natural Language Processing (NLP) and Streamlit. It predicts whether a movie review is **Positive** or **Negative** using a Machine Learning model trained on the IMDB Movie Reviews dataset.

## 🚀 Features
- Predicts Positive or Negative sentiment
- Text preprocessing using NLP
- TF-IDF Vectorization
- Machine Learning Classification
- Interactive Streamlit Web Application
- Fast and User-Friendly Interface

## 🛠️ Technologies Used
- Python
- NLP
- Scikit-learn
- Pandas
- NumPy
- Streamlit
- Pickle



## ▶️ Run the Project

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 📊 Dataset
- IMDB Movie Review Dataset
